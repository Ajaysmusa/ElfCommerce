# LF Commerce

An open source ecommerce system front-end written in ReactJS

This is for the missing piece of ExpressJS + ReactJS solution for ecommerce system.

![Alt Screenshot](./screenshot.png "Screenshot")


## Installation

**Yarn**
```console
yarn install
```

**NPM**

```console
npm install
```


## How to run this?

**Yarn**

```console
yarn start
```

**NPM**

```console
npm start
```
